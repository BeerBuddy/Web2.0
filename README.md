# Web2.0

##About
This is a AngularJS + Spring Boot Project with  [J-Hipster](https://jhipster.github.io/)

## Getting started
1. Install NodeJS from: [https://nodejs.org](https://nodejs.org)
2. Install [IDEA Ultimate](https://www.jetbrains.com/idea/download/) [Lizens](https://www.jetbrains.com/student/)
3. Checkout Project
4. Open Project with Webstorm
6. **File** > **Settings** > **Plugins**
	* NodeJs, AngularJS, Jade, Karama und LiveEdit
7. Restart IDE
8. **File** > **Settings** > **Node.JS and NPM**
 	* NodeJS Interprtor auswählen
	* Module Yo, grunt-cli, bower und generator-jhipster hinzufügen
	** Für JHipster musste ich im Terminal ein 'npm install -g generator-jhipster' absetzen.
9. **File** > **Settings** > **Bower**
	* bower.json im Projekt auswählen
	* NodeJs Interpreter auswählen
	* Bower auswählen Bei mir wars unter C:\Users\David\AppData\Roaming\npm\node_modules\bower\bin\bower
10. Restart IDE
11. 'build.gradle' mit Parameter 'bootRun' starten
